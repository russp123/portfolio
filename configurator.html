Kingdomes Product Configurator

Interactive Web App

A standalone HTML/JavaScript application built for Kingdomes, featuring procedurally generated SVG dome previews, a carousel state machine with selection-hierarchy logic, and real-time pricing across 5 dome sizes. It integrates with GoHighLevel via webhook for direct lead capture, and has been iterated across 8+ build versions. Not yet launched publicly — this is a completed build, not a live product with usage metrics.

Pricing Carousel

A carousel state machine drives selection across 5 dome sizes, recalculating price in real time as options change, with selection-hierarchy logic that keeps incompatible combinations from being chosen.

Lead Capture

Once a configuration is finalized, the app hands off to GoHighLevel via webhook for direct lead capture — no manual data entry between the configurator and the CRM.
